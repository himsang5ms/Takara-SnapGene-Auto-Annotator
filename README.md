# 🧬 Takara to SnapGene Auto-Annotator 
**—— 拯救生化环材狗的克隆效率外挂**

在做 In-Fusion 无缝克隆时，你是否受够了在 SnapGene 里一段段手动添加引物、核对酶切位点？手动标注不仅废眼，还极容易发生移码错配，导致几个月的实验打水漂。

本项目提供了一个极简的自动化脚本，完美桥接 Takara 官网设计与 SnapGene 可视化图谱。

## ✨ 核心功能 (Features)
* 🚀 **一键导出：** 直接将 Takara 网页端的引物设计结果导出为标准的 `.gb` 格式文件。
* 🎯 **全自动注释：** 导出的文件完美兼容 SnapGene，自动标注引物位置、和目的片段。
* 🛡️ **零错配风险：** 机器自动转换，彻底杜绝人工复制粘贴导致的碱基遗漏或移码。

## 📥 安装与使用 (Installation)

1. 浏览器安装 Tampermonkey 插件。
2. 点击本仓库的 Takara_to_SnapGene.js 文件，点击右上角的 Raw 按钮，油猴会自动弹出安装界面。
3. 安装完成后，打开 Takara In-Fusion 页面即可生效！
